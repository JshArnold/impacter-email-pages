// PROCESS — FOUR_STAGE_PIPELINE
01
01_INTAKE
Student Speaks

Student responds to a domain-specific voice prompt via VideoAsk. No login required. Anonymous by default. Works on any device with a microphone.

02
02_NLP_SCORE
NLP Analysis

Transcript is extracted and analyzed against six behavioral health domains. Each domain scored 0–4. Tier placement is computed in under 5 seconds.

03
03_MTSS_ASSIGN
MTSS Placement

Score maps to Tier 1, 2, or 3. Aggregate cohort data surfaces patterns across classrooms, grade levels, and schools — without individual student exposure.

04
04_DASHBOARD
Educator View

Counselors and administrators access real-time dashboards. Filter by domain, grade, cohort. Export for MTSS documentation. Reimbursement-ready reporting.

Through California's Children and Youth Behavioral Health Initiative (CYBHI), districts can bill Medi-Cal for each screener administered using CPT code 96127. At $4.81 per screen, the per-student cost of running IMPACTER is fully offset — in many cases generating a surplus that funds additional support services.
